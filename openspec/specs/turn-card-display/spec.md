# turn-card-display Specification

## Purpose

**Superseded by `turn-card-v2`** (see `openspec/specs/turn-card-v2/spec.md`).

This capability described the original turn card layout. The `turn-card-v2-expert-redesign` change replaced the entire layout (five-line layout, left-edge severity bar, plain-ASCII risk markers, cache composition bar, etc.) and the `turn-card-display-amendments` change tweaked the v2 result. All requirements that previously lived here are now in `turn-card-v2`. The two observability badges (`sys-changed`, `thinking-stripped`) added by the `p0-observability-signals` change are likewise on `turn-card-v2` line 5.

This stub is retained so historical archive references in `openspec/changes/archive/` continue to resolve to a known capability path.

## Requirements

_None — see `turn-card-v2` for the active spec._
